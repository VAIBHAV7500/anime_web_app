# Installation

## Requirements

* Yarn - Setup yarn from [here](https://classic.yarnpkg.com/en/docs/install/#windows-stable)

* Node - Setup Node from [here](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

## Backend:
* Install packages required.

    ```
    cd backend
    npm install
    ```

## Frontend
* Install packages required.
    ```
    cd frontend
    npm install
    yarn install
    ```
